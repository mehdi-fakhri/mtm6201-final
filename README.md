# MTM6201 Final Project – Web Portfolio

## Project Overview
This is my final project for MTM6201, showcasing my skills in HTML, CSS, Bootstrap, responsive design, and accessibility.

## My Process
- Started with the **index page**, designing layout and colors.  
- Built the **second page (adopt)** and **third page (involved)** based on the index layout.  
- Used **Bootstrap** and **CSS variables** for customization.  
- Learned more about **media queries** to make the site responsive on all devices.  
- Added hover effects, interactive elements, and ensured accessibility with alt text and semantic HTML.  

## Lessons Learned
- Improved responsive design and accessibility skills.  
- Learned to integrate Bootstrap while customizing styles.  
- Learned to troubleshoot layout and interactive issues.
